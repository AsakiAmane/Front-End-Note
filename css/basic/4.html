CSS(Cascading style sheets 层叠样式表)

作用：决定样式美观

像现在，由于css里指定了p的样式使用cadetblue这个颜色，这段p标签的内容全是指定的颜色

最简单的一种写法是内嵌式：写在位于head标签内，title标签下的style标签中

外链式：css单独写在一个css文件中，通过link标签在网页中引入

项目中一般用外链式，而配合js使用的情况采用行内式（使用概率很低），而小案例则可以是内嵌式

行内式：css写在标签的style属性中，一般配合JavaScript使用

但如果是父标签中的子标签已经受link的css文件指定样式，则一般不受父标签的style影响

h6为指定样式的div的子标签且link的css文件没指定样式，故不受影响
子标签内style也是与父标签单独的

而如果link的css已经指定一个标签样式，但html文件中再在某个同种标签中指定样式，则会优先html的

选择器

标签选择器：

标签名{属性}，如这个文件里的内嵌css，但是标签选择器会让所有同种标签都生效

类选择器：

.类名{属性}，然后在标签中标注该标签是什么类，注意：标签中的类不用带'.', 复杂的情况下一个标签可以有多个类名，类名之间用空格隔开

id选择器：

#id{属性}，id初衷用于js，所以很少用来做样式, id类似身份证号码，一个标签只有一个id，一个选择器规定上只能选中一个标签

通配符选择器：

*{属性}，*代表全选, 一般只用于清楚默认的内外边距，如margin:0; padding:0;

字体和文本样式

字体样式：

1. 字体大小 font-size；chrome默认文字大小为16px

2. 字体粗细 font-weight；常用normal和bold，也可以直接用整百数（一般常用，用400和700）, 可以留意到整数后不用单位

3. 字体样式是否倾斜 font-style，默认值normal，倾斜为italic, 倾斜标签也有个是i

4. 字体风格 font-family，属性值写字体，一般用默认的字体，微软是微软雅黑，苹果是苹方，两者都是无衬线字体,在font-family的设置中按属性值的先后确定优先级（sans-serif）

5. 复合属性简写：上述样式可以直接用font复合
