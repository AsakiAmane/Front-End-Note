元素（即标签）显示模式

显示模式：即一行几个等
元素类型：块级、行内、行内块

1. 块级元素

独占一行（如div、hx、p、ul、li、dt、dd、form等）、宽度默认是父元素的宽度，高度默认由内容撑开、但可以设置宽高，一般作为大容器，可以嵌套（但p不要嵌套div、p、h等块级元素，a不能嵌套a标签）

2. 行内元素

一行可以显示多个（a、span、b、u、i、s...），宽度高度默认由内容撑开，不可设置宽高

3. 行内块元素

一行可以显示多个且可设置宽高（input、textarea、button、select、img...）

4. 显示模式的转换

改变元素默认的显示特点使其符合布局要求
display: block 转换为块级元素 使用频繁
display: inline-block 转换为行内块元素 使用频繁
display: inline 转换为行内元素 使用极少
上面一行是两个div进行了行内块元素转换
